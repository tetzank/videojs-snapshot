This is a demo page of the videojs snapshot plugin. Some hints how to use it:

  1. Choose a local mp4 file.
  2. Press at any position in the video the camera symbol on the control bar. This opens the still image for editing.
  3. Edit the image with the provided tools at the bottom. Read the tooltips and play a bit around.
  4. When you're done, click on JPEG or PNG to encode the image.
  5. Rightclick and save the encoded image.
